CSS

CSS é a linguagem de estilização usada junto ao HTML

Estrutura básica do CSS

seletor {
propriedade: valor; 
}

Seletores

O seletor se refere a um ou mais elementos utilizando uma tag, classe, id ou pseudo classe, podendo se usar mais de um seletor por estrutura, e até mesmo combinar seletores em um só para especificar ainda mais que elementos serão selecionados pelo seletor combinado, além de também poder selecionar se baseando na estrutura de pai/filho do HTML

Selector Specificity

A quantidade de seletores combinados em um resulta no valor de prioridade que a seleção tem sobre outras, sendo calculada por 3 variáveis que formam o número da prioridade, onde o número mais alto tem prioridade sobre os mais baixos

Essa prioridade é calculada pela Selector Specificity(podemos ver ela ao passar o mouse em cima do seletor) e se dá por pontuação de acordo com a combinação de seletores, onde cada tag vale uma unidade de pontos, cada classe vale uma dezena e ids valem uma centena
