Importando fontes

Google Fonts

Usando o site google fonts você pode fazer importações facilitadas ao selecionar a opção de obter o código embutido(Get embed code)

Tal opção de dá o código necessário para importar a fonte diretemnte do CSS, sem ter que baixar nada, e reduzindo a necessidade do usuário ter conhecimento prévio sobre o código

Podemos importar a partir de uma tag link no HTML(não testado) ou de um @import no CSS(caso de uso atual)

Uma vez que a fonte foi importada podemos usa-la no CSS, utilizando a propriedade font-family: podemos escolher a fonte importada como valor ao colocar seu nome entre ""(aspas duplas).

Notas do autor

Como curiosidade, me parece que devemos adicionar uma segunda fonte na font-family como uma boa prática, suspeito que para caso de falha na importação. Também não observei mudanças práticas ao usar aspas simples ou não usar aspas(nesse caso só a mudança de cor no vs code) no nome da fonte durante a definição do valor da propriedade font-family
