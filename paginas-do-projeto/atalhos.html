Atalhos

A utilização de atalhos é crucial para aglizar do desenvolvimento e manutenção do código, aqui constam os mais relevantes que encontrei

Atalhos no Visual Studio Code

Ctrl+D

Usado para editar as partes iguais a selecionada no código ao mesmo tempo, muito útil para mudar valores de propriedades no CSS, ou alterar nomes de classes e nomes no HTML

Ctrl+K+C

comenta a linha atual do código
