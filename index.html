Sobre o projeto

Este site visa exibir material informativo e auto-explicativo sobre a construção de uma página web com HTML, CSS e Javascript. Tem como seu objetivo principal servir de material de revisão para oque aprendi mas que costumo esquecer caso não utilize por um tempo. Cada página será dedicada a explicações sobre um determinado tema específico e poderá conter também alguns temas menores que estão relacionados.

Objetivo final

Com esse projeto espero criar um site documentacional simulando o MDN Web docs da Mozilla, mas com conteúdo informal, mais resumido, e menos denso, voltado para ser uma tira dúvidas de assuntos já visitados e usados, talvez também úteis para uma primeira olhada de um leigo no assunto.
